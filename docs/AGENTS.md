# AGENTS.md (Docs)

## Scope
이 파일은 `docs/` 하위 문서에 적용됩니다.

## Documentation Policy
- PRD 기준 기능 변경 시 `API.md`, `ARCHITECTURE.md`를 함께 갱신합니다.
- 날짜는 절대 날짜(YYYY-MM-DD)로 표기합니다.
- 오락용 면책/개인정보 삭제 정책 문구를 문서마다 일관되게 유지합니다.

## Minimal Required Docs
- `PRD.md`: 제품 요구사항
- `API.md`: 엔드포인트, 요청/응답, 오류 코드
- `ARCHITECTURE.md`: 클라이언트/서버/저장소 구성
